1. Energy Assessment Web Map for CATF (Africa)

Date: April 2025

Context & Goal: Built an interactive map displaying energy infrastructure and assessment metrics across African countries for the Clean Air Task Force.

Tech Stack: Uses Python for basic data wrangling, stores spatial data as GeoJSON, renders with Mapbox GL JS, and is built with standard HTML, CSS, and JavaScript.

Key Features: Country‑level energy scorecards, selectable layers, pop‑ups with energy data, responsive legend.

Outcome: Gave CATF a template to explore regional performance and identify investment priorities for future web applications. View map ↗ | View repository ↗
